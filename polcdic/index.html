プレイオンラインにて配布されていたMicrosoft IME用の辞書と互換性のある辞書です。

辞書ファイル

辞書名 ファイル名 用途

polcdic-kaomoji.txt Google 日本語入力用
POL互換辞書（FF11感情表現コマンド） polcdic-ff11emote.dctx Microsoft IME用
polcdic-ff11emote.txt Google 日本語入力用

システム要件

  • Microsoft IME Windows 8.1
  • Microsoft IME Windows 10
  • Google 日本語入力

使い方（Microsoft IME Windows 8.1/Windows 10）

  1. zipファイルを展開する。
  2. polcdic-kaomoji.dctx をダブルクリックする。
  3. 「この辞書を使用する場合は[OK]を、」と表示されたらOKを押す。
  4. トレイアイコンの IME を右クリック「追加辞書サービス」＞「辞書の設定」を開く。
  5. 「POL互換辞書（顔文字）」にチェックを入れ、OKを押す。
  6. 完了したら.dctxファイルは削除しても良い。

使い方（Google日本語入力）

  1. zipファイルを展開する。
  2. Google日本語入力のトレイアイコンを右クリックし「辞書ツール」を開く。
  3. 「管理」＞「新規辞書にインポート」を押す。
  4. 「ファイルを選択」を押し「polcdic-kaomoji.txt」を選択する。
  5. 「辞書名」に「POL互換辞書（顔文字）」と入力する。
  6. 「インポート」を押す。
